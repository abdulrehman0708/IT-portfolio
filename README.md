# IT Portfolio – Dark React Template

A dark, elegant personal portfolio template built with **React (Vite)**.  
Designed for IT professionals, developers, and system admins.

## 🚀 Features
- Personal intro and about section  
- Skills, projects, and contact page  
- Fully responsive layout  
- Simple and fast loading

## 🛠️ Tech Stack
- React (Vite)
- TailwindCSS
- React Icons

## ⚙️ Run Locally
1. `npm install`
2. `npm run dev`
